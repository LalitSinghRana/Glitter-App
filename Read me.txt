#Glitter App

A twitter like web app created using 
*	dotNet MVC web API as backend (3 layered architecture)
*	Angular as front end
*	and MS SQL server as database


App Setup

•	The production server for UI runs at http://localhost:4200/ and web api at https://localhost:44385 
•	Change database connection string name in backend(web API) to your local server's name 

How to Run the App?
-	Download node_modules for angular first.
-	Then "ng-serve" command from terminal for UI
-	Web-Api is run from the Presentation startup project of the Solution
